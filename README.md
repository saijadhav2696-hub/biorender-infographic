Infographic: CRISPR-Cas9 Gene Editing Mechanism

Overview

This infographic represents the CRISPR-Cas9 gene editing mechanism. It visually explains how the Cas9 protein and guide RNA identify a target DNA sequence, introduce a double-strand break, and how the cell repairs it through NHEJ (Non-Homologous End Joining) or HDR (Homology-Directed Repair) pathways.

Key Steps

- Target DNA recognition
- Binding of Cas9-gRNA complex
- DNA cleavage (double-strand break)
- DNA repair via NHEJ or HDR
- Formation of edited gene

Tools Used

- BioRender (for infographic design)

Description

This work was created as part of an assignment to convert a research protocol into a visual infographic for better understanding of complex biological processes.
